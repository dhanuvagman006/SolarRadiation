MAE (Mean Absolute Error) Average absolute deviation in kWh/m²

🔑 Input Features Description

T2M - Temperature (°C)

Range: 15-35°C

Air temperature at 2 meters. Higher temp generally means clearer skies.

RH2M - Relative Humidity (%)

Range: 40-95%

Water vapor in air. High humidity indicates cloud formation.

WS2M - Wind Speed (m/s)

Range: 0.5-4.0 m/s

Wind speed affects cloud movement and atmospheric clarity.

CLOUD_AMT - Cloud Amount (%)

Range: 0-100%

Cloud cover percentage. Primary factor affecting solar radiation.

PRECTOTCORR - Precipitation (mm)

Range: 0-10 mm

Daily precipitation. Indicates weather severity.

Temporal Features

day, month, year 1-31, 1-12, YYYY

Captures seasonal and annual patterns.
